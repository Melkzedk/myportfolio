
# My Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and experience as a developer.

## Features
- Responsive design using React and Tailwind CSS
- Navigation bar and footer components
- Dedicated pages for About, Contact, Projects, Resume, and Skills
- Downloadable resume
- Image gallery

## Technologies Used
- React
- Tailwind CSS
- JavaScript
- PostCSS

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

### Installation
1. Clone the repository:
	```powershell
	git clone https://github.com/Melkzedk/myportfolio.git
	```
2. Navigate to the project directory:
	```powershell
	cd myportfolio
	```
3. Install dependencies:
	```powershell
	npm install
	```

### Running the App
Start the development server:
```powershell
npm start
```
The app will be available at `http://localhost:3000`.
